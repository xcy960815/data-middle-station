# DAO 类型定义

本目录包含应用程序中数据访问层 (DAO) 的 TypeScript 类型定义 (`.d.ts` 文件)。这些定义描述了与数据库交互的数据结构。

## 目录结构

| 文件名                      | 描述                                                                                                                 |
| --------------------------- | -------------------------------------------------------------------------------------------------------------------- |
| `AnalyzeConfigDao.d.ts`     | 定义图表配置的类型，包括列配置、维度、过滤条件、分组、排序以及各类图表（折线图、柱状图、饼图、表格）的具体配置选项。 |
| `AnalyzeDao.d.ts`           | 定义分析（Analyze）实体的基础数据结构，通常对应数据库中的分析表。                                                    |
| `AnalyzeDataDao.d.ts`       | 定义分析数据的类型，用于描述图表查询返回的具体数据结构。                                                             |
| `DataBaseDao.d.ts`          | 定义数据库元数据的类型，包括表结构信息（表名、注释、行数等）和字段信息（列名、类型、注释）。                         |
| `LoginDao.d.ts`             | 定义登录相关的数据结构，如登录请求参数等。                                                                           |
| `ScheduledEmailDao.d.ts`    | 定义定时邮件任务的数据结构，包括任务的基本信息、调度配置、邮件内容配置、任务状态及重试策略等。                       |
| `ScheduledEmailLogDao.d.ts` | 定义定时邮件任务执行日志的数据结构，用于记录每次任务执行的状态、结果和错误信息。                                     |
| `SendEmailDao.d.ts`         | 定义发送邮件操作相关的数据结构，包括邮件发送的请求参数配置。                                                         |

## 使用说明

这些命名空间是全局声明的，主要用于 Server 端 Service 层和 Mapper 层的数据交互，确保数据库操作的类型安全。

示例：

```typescript
// 在 Service 或 Mapper 中使用
const chartConfig: AnalyzeConfigDao.ChartConfigOptions = {
  // ...
}
```
